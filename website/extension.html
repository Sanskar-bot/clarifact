Installation

Load it in four steps.

ClariFact isn't on the Chrome Web Store yet — it's distributed as an unpacked extension. This takes about 60 seconds and requires no account.

01

Download the extension ZIP

Download the latest release from the downloads page or the GitHub releases page. Extract the ZIP to a folder you'll remember.

02

Open Chrome Extensions

In Chrome, navigate to chrome://extensions. Enable Developer mode using the toggle in the top-right corner.

03

Load the unpacked folder

Click Load unpacked, then select the extracted extension folder. ClariFact will appear in your extensions list.

04

Select text and check

On any webpage, highlight text, right-click, and choose ClariFact: Check this claim. The verdict appears in a sidebar on the same page.

Note on updates
Because ClariFact isn't on the Chrome Web Store, updates don't install automatically. When a new version is released, download the new ZIP, replace the extracted folder contents, and return to chrome://extensions to click the refresh icon next to ClariFact.
Permissions & Privacy

What the extension can see — and why.

Extensions that can read page content are worth scrutinising. Here's exactly what ClariFact requests and why.

Permission Why it's needed
contextMenus Required to add the right-click "ClariFact: Check this claim" option to the browser's context menu.
activeTab Required to read the text you've selected on the current tab. Only the selected text is read — not the full page content, not your browsing history.
storage Used to store your extension preferences (sidebar position, result history within the current session) locally in your browser. Nothing is synced externally.
scripting Required to inject the sidebar UI into the current tab when a check is triggered.
host_permissions (api) Needed to call the ClariFact backend API at clarifact.onrender.com to process the claim. No other external hosts are contacted.
What is never read
The rest of the page you're on. Your browsing history. Other tabs. Cookies. Passwords or form data. Only the text you explicitly select and choose to fact-check is ever sent anywhere.

The complete extension source code is on GitHub — you can read every line before installing.
