What it does

Fact-check anything, inline

ClariFact adds a permanent fact-checking layer to your browser. Select text, get a verdict. No tab-switching, no manual googling.

Instant Fact-Check
Select any text, click the floating button. ClariFact runs a full search-and-analyse pipeline and shows results in a sidebar — no new tab, no copy-paste.
AI-Powered Verdict
Nemotron Nano 3 30B (via AWS Bedrock) reads the top sources and returns a structured SUPPORTED / INCONCLUSIVE / CONTRADICTED verdict with confidence score.
PDF Source Support
Government reports, court orders, and official documents in PDF format are automatically detected and parsed — the most authoritative sources are no longer skipped.
Source Cards with Trust Scores
Every source is shown as an expandable card with domain trust score, the best-matching sentence, and a direct link. PDF sources get a dedicated badge.
Contradiction Detection
When sources disagree with the claim, contradictions are surfaced separately — so you can see not just the verdict, but exactly what disputes it.
Copy Report
One click exports a plain-text fact-check report — claim, verdict, confidence breakdown, all source URLs, and AI explanation — ready to paste anywhere.
Installation

Up and running in under 5 minutes

ClariFact isn't on the Chrome Web Store yet, so installation takes one extra step — but it's quick. No developer tools, no command line, no API keys. Just download, extract, and load.

1
Download the Extension ZIP
Click the button below to download the ClariFact extension as a ZIP file. No installer or build step required — the ZIP contains the complete, ready-to-load extension.
clarifact-extension-v1.1.0.zip
2
Extract the ZIP
Right-click the downloaded file → Extract All (Windows) or double-click (macOS). Note where the extracted folder is — you'll need it in Step 6.
The extracted folder should contain a manifest.json file at its root. That's the folder you'll point Chrome to.
3
Open Chrome (or Edge / Brave / Opera)
ClariFact supports any Chromium-based browser. All use the same extension package.
4
Go to the Extensions page
In the address bar, type and press Enter:
chrome://extensions
For Edge: edge://extensions   For Brave: brave://extensions
5
Turn on Developer Mode
Look for the Developer mode toggle in the top-right corner of the Extensions page and switch it ON. A few extra buttons will appear at the top — that's expected.
This is just Chrome's way of allowing extensions that aren't in the Web Store. It doesn't make your browser less safe.
6
Click "Load Unpacked"
Click the Load unpacked button that appeared after enabling Developer mode. A file picker dialog will open.
7
Select the ClariFact Extension Folder
Navigate to the folder you extracted in Step 2. Select the folder that contains manifest.json and click Select Folder.
Select the folder itself, not any file inside it. The folder should be named something like extension or clarifact-extension-v1.1.0.
That's it — you're done!
ClariFact is installed and connected to the cloud backend automatically. Go to any webpage, highlight a sentence or claim, and click the Fact-check button that appears. You'll see a verdict, evidence, and sources slide in — no extra setup needed.
The backend runs 24/7 on a hosted server. Your first fact-check might take ~30 seconds if the server is waking from idle — subsequent ones are fast.

Transparency

Permissions — and why each one is needed

ClariFact requests the minimum permissions required. Here is exactly what each one does.

activeTab
Inject the sidebar and floating button into the current tab. Only activated when you explicitly trigger a fact-check — not on page load.
scripting
Required by Chrome MV3 to programmatically inject content scripts when triggered by the context menu or keyboard shortcut.
storage
Stores the last fact-check result locally so the popup can display it when you click the toolbar icon. Nothing is synced to any server.
contextMenus
Adds a "Fact-check with ClariFact" right-click menu option as an alternative to the floating button.
host: clarifact.onrender.com
Allows the extension to talk to the ClariFact cloud backend. This is the only external connection the extension makes — it sends your selected text to get a fact-check result back. No other sites are contacted.
Privacy & Security

What happens to your data?

Short answer: very little. ClariFact sends only the text you highlight — nothing else. Here's the full picture, in plain language.

Privacy by design
Not a policy — verifiable from the source code
Only the highlighted text is sent. When you fact-check something, just that claim goes to the backend — not the URL, not the page, not your history.
Nothing is stored on the server. The backend is stateless — it processes each request and discards it. No database, no user accounts, no logs of your claims.
No tracking or analytics. The extension has zero analytics, telemetry, or crash-reporting code — confirmed by reading the source.
Results stay on your device. Fact-check results are saved in your browser's local storage — they never leave your machine.
Fully open source. Every line — extension and backend — is on GitHub. No black boxes.
Minimal permissions. Only 4 Chrome permissions — each one explained in the section above.
