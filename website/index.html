Features

Everything you need to
stop believing false things.

Free, no account, no setup. Just install and start checking.

Searches the Web For You

The moment you highlight text, ClariFact searches the web and finds the most relevant sources — news sites, government pages, Wikipedia, and more.

Actually Reads the Articles

It doesn't just look at headlines — it reads the full article text of each source, so the verdict is based on real content, not titles.

AI Gives a Clear Verdict

An AI model reads all the sources and tells you: Supported, Contradicted, or Inconclusive — with a plain-English explanation of why.

Flags What Contradicts the Claim

If any source says the opposite of the claim, ClariFact pulls out that exact sentence and shows you — so you can read it yourself.

Shows How Confident It Is

A confidence score (0–100%) shows how strongly the sources back up the verdict. High score = many good sources agree. Low score = mixed or weak evidence.

Your Data Stays Private

Only the text you highlight is sent for analysis. No browsing history, no page content, no tracking. Results are saved only on your own device.

Links to Every Source

Every result shows the actual articles it used — with the relevant quote highlighted. You can click through and read the original yourself.

Copy & Share the Report

One click copies the full fact-check — claim, verdict, sources, and AI explanation — as plain text. Share it in a chat, email, or document.

How It Works

From highlight to verdict
in under 5 seconds.

01

You highlight text

Select any sentence or claim on any webpage. A small Fact-check button pops up. Click it — or right-click and choose ClariFact.

→
02

We search the web

ClariFact searches the web for the most relevant sources — news articles, Wikipedia, government sites, and more — all related to your claim.

→
03

We read the articles

The full text of the top sources is read automatically. Paywalled or blocked sites are skipped. Only the real article content is used.

→
04

AI forms a verdict

An AI model reads all the source content alongside your claim, and decides: Supported, Contradicted, or Inconclusive — with evidence to back it up.

→
05

We calculate confidence

A confidence score is calculated based on how many sources agree, how trustworthy they are, and how closely they match the claim.

→
06

You see the verdict

A sidebar slides open with the verdict, AI explanation, confidence score, source links, and any contradictions found. Takes about 4 seconds total.

Get Started

Installed in
30 seconds.

No accounts. No API keys. Just download and start fact-checking.

1

Download the extension

Click the button below to get the ClariFact extension as a ZIP file. No build tools or Node.js required.

Download clarifact-extension-v1.1.0.zip

ZIP · 1.3 MB · No installer required

2

Load it in Chrome

Extract the ZIP, then:

  1. Go to chrome://extensions in the address bar
  2. Toggle Developer mode ON (top-right corner)
  3. Click Load unpacked → select the extracted folder (the one containing manifest.json)
Need more detail? See the full installation guide →
3

Select any text and click Fact-check

Navigate to any webpage, highlight a claim, and click the Fact-check button that appears. The hosted backend handles search and AI analysis automatically — you'll see a full verdict in the sidebar within seconds.

✓ Nothing to configure. The backend runs 24/7 on a server I maintain, rate-limited per IP. Prefer to self-host with your own keys? Full source is on GitHub.
Under the Hood

Three things working
together for you.

You don't need to know any of this to use it — but here's what's happening behind the scenes.

🔍

Web Search

ClariFact searches the web using Tavily — a search engine built for AI tools. It finds diverse, relevant sources quickly, not just the most clicked links.

Always on
🤖

AI Analysis

An AI model (Nemotron, by NVIDIA) reads all the source content and your claim together. It only uses what it read — it can't make things up.

Cloud · Free for you
🌐

Chrome Extension

The extension runs inside your browser. It watches for what you highlight, sends it for analysis, and shows the result — without ever opening a new tab.

Open source · MIT
FAQ

Common questions.

When you trigger a fact-check, the selected claim text and a list of source URLs are sent to the ClariFact hosted backend (on Render). The backend calls Tavily Search and Nemotron AI (via AWS Bedrock) to analyse the claim.

What is NOT sent: your browsing history, cookies, full page HTML, or anything outside the text you highlighted. There are no user accounts and no persistent logs of your queries.

It works on most websites. Chrome restricts content scripts from running on chrome:// pages, the Chrome Web Store, and certain internal pages. Hard-paywalled sources (WSJ, FT) are automatically skipped during scraping. JavaScript-rendered SPAs may return incomplete text since the backend fetches raw HTML.

Free. The hosted backend absorbs the Tavily and AWS Bedrock API costs. Per-IP rate limiting keeps usage fair for everyone. If you'd prefer to run your own backend with your own keys, the full source code is on GitHub.

No. The backend runs 24/7 on Render's cloud infrastructure — just install the extension and it connects automatically. The popup shows a green dot when the backend is reachable and a red dot with a retry button if it's temporarily unreachable (e.g. during a cold start on the free tier, which can take ~30 seconds).

ClariFact is a research aid, not a final authority. It tells you what the top web sources say about a claim — and how confidently. For breaking news or niche topics, sources may be limited or conflicting. Always read the linked articles yourself before drawing conclusions.

Mostly yes. The web search and AI analysis work across many languages. Entity detection (identifying names, places, dates in the claim) is English-optimised, so confidence scores may be slightly lower for non-English claims. A note is shown in the sidebar when this applies.
