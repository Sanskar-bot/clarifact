Features

Everything you need to
stop misinformation.

No subscriptions. No data collection. Runs entirely on your machine.

Brave Search API

Queries Brave's independent search index (not Google) to get unbiased, real web results for any claim. API key is kept server-side — never exposed.

Readability.js Scraping

Uses Mozilla's Readability.js to extract clean article text from the top 5 search results — stripping ads, nav, and noise. Runs server-side to bypass CORS.

Claude 3.5 Sonnet AI

Amazon Bedrock powers a grounded AI analysis. Claude reads the scraped sources and returns a verdict, explanation, and key evidence — based only on what it read.

compromise.js NLP

Extracts named entities (people, places, organisations, dates) from the claim using compromise.js — all on-device. No cloud NLP service required.

Contradiction Detection

Scans source sentences for negation patterns that directly contradict the claim. Surfaces specific contradicting sentences with their source domain.

Zero Data Storage

The backend binds only to 127.0.0.1 — never reachable from the internet. Every request is stateless. Nothing is logged, stored, or sent to third parties.

4-Signal Confidence Score

Combines source count (20%), domain trust (25%), entity match (30%), and text similarity (25%) into a single 0–100% NLP confidence score.

50+ Trusted Domains

A curated trust database rates domains from 0 to 1 — NASA, Reuters, and academic journals score near 1.0; unknown blogs score near 0.4. Affects the final score.

Pipeline

From selection to verdict
in under 5 seconds.

01

Select & Trigger

Highlight any text on any webpage. A floating Fact-check button appears. Click it — or right-click and choose Clarifact from the context menu.

→
02

Brave Search

The backend proxies a query to Brave's independent search index, retrieves the top 10 results, and deduplicates by domain to ensure source diversity.

→
03

Parallel Scraping

The top 5 unique sources are fetched and parsed with Readability.js in parallel, extracting clean article text. Paywalled or bot-blocked sources are automatically skipped.

→
04

Claude 3.5 AI

Source excerpts + the claim are sent to Claude 3.5 Sonnet via Amazon Bedrock. Claude returns a structured JSON verdict grounded exclusively in the scraped sources.

→
05

NLP + Scoring

On-device: compromise.js extracts entities, TF-cosine finds best-matching sentences, negation scanning detects contradictions. A 4-signal score is computed.

→
06

Sidebar Result

A beautiful in-page sidebar slides in with the verdict, confidence score, AI explanation, entity tags, source cards, contradictions, and a copyable report.

2

Load it in Chrome

Extract the ZIP, then:

  1. Go to chrome://extensions in the address bar
  2. Toggle Developer mode ON (top-right corner)
  3. Click Load unpacked → select the extracted folder (the one containing manifest.json)

FAQ

Common questions.

When you trigger a fact-check, the selected claim text and a list of source URLs are sent to the ClariFact hosted backend (on Render). The backend calls Tavily Search and Nemotron AI (via AWS Bedrock) to analyse the claim.

What is NOT sent: your browsing history, cookies, full page HTML, or anything outside the text you highlighted. There are no user accounts and no persistent logs of your queries.

It works on most websites. Chrome restricts content scripts from running on chrome:// pages, the Chrome Web Store, and certain internal pages. Hard-paywalled sources (WSJ, FT) are automatically skipped during scraping. JavaScript-rendered SPAs may return incomplete text since the backend fetches raw HTML.

Free. The hosted backend absorbs the Tavily and AWS Bedrock API costs. Per-IP rate limiting keeps usage fair for everyone. If you'd prefer to run your own backend with your own keys, the full source code is on GitHub.

No. The backend runs 24/7 on Render's cloud infrastructure — just install the extension and it connects automatically. The popup shows a green dot when the backend is reachable and a red dot with a retry button if it's temporarily unreachable (e.g. during a cold start on the free tier, which can take ~30 seconds).

The hosted backend uses IAM credentials (AWS_ACCESS_KEY_ID + AWS_SECRET_ACCESS_KEY) with SigV4 signing to call Amazon Bedrock. If you self-host, you can also use AWS_BEARER_TOKEN_BEDROCK for bearer token auth — the backend auto-detects which method you've configured. You only need one of the two.

The extension degrades gracefully. The NLP pipeline (compromise.js + TF cosine + negation detection) always runs. If Bedrock fails or is not configured, the AI Analysis section in the sidebar shows "Not configured" instead of a verdict. The NLP-only result is still displayed in full.

On the hosted backend, the domain trust list is managed centrally. If you self-host, edit backend/data/trustedDomains.json — add any domain with a trust score between 0 and 1 (e.g. "pubmed.ncbi.nlm.nih.gov": 0.97) and restart the server. No code changes required.

Partially. Tavily Search returns results for any language. Readability.js extracts text in any language. However, compromise.js's entity extraction is English-only, reducing NLP accuracy for non-English claims. Nemotron Nano 3 30B handles many languages for the AI analysis stage. A warning is shown in the sidebar when a non-English claim is detected.
